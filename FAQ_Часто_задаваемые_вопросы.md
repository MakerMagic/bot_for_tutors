## 🔐 Безопасность

### Безопасно ли хранить данные учеников?

**Да!** Вот что хранится:
- Telegram ID (число)
- Username (публичный)
- Имя (которое ученик сам указал в Telegram)
- Информация о занятиях и оплате

**НЕ хранится:**
- Номера телефонов
- Адреса
- Паспортные данные
- Банковские карты

### Кто имеет доступ к данным?

**Только ты!** База данных находится на твоём аккаунте PythonAnywhere.

Никто кроме тебя не может получить доступ к данным учеников.

### Что если потеряю токен бота?

1. Зайди в Telegram → [@BotFather](https://t.me/BotFather)
2. Отправь `/mybots`
3. Выбери своего бота
4. Нажми **API Token**
5. Скопируй токен
6. Замени в файле `complete_bot.py`
7. Перезапусти бота

## 💾 База данных

### Где хранятся данные?

В файле `tutor_bot.db` на твоём аккаунте PythonAnywhere.

### Как сделать резервную копию?

1. Зайди на PythonAnywhere
2. Открой вкладку **Files**
3. Найди файл `tutor_bot.db`
4. Нажми **Download**
5. Сохрани с датой: `tutor_bot_2026_02_16.db`

**Делай копии раз в месяц!**

### Можно ли перенести базу на другой компьютер?

**Да!** Просто скопируй файл `tutor_bot.db`.

Положи его в ту же папку где `complete_bot.py` и `main.py` - всё заработает.

### Как очистить базу и начать заново?

**Вариант 1 (удалить всех учеников, но сохранить структуру):**
```bash
rm tutor_bot.db
python3.10 main.py
```

**Вариант 2 (удалить конкретного ученика):**
```
/remove Никнейм
```

## 💰 Оплата

### Как работает подсчет оплаты?

Бот считает **по реальным дням занятий**, а не календарным месяцам.

**Пример:**
- Ученик занимается по вторникам и пятницам
- Оплатил 8 занятий
- Бот найдет 8 ближайших занятий по расписанию
- Последнее (8-е) занятие = "Оплачено до"
- Следующее (9-е) занятие = "Следующая оплата"

### Что если у ученика нет постоянного расписания?

Команда `/addpayment` **не сработает**.

Сначала добавь расписание:
```
/addschedule Никнейм день время
```

### Можно ли вручную изменить дату оплаты?

**Нет.** Бот считает автоматически.

Если нужно - используй `/clearpayment` и добавь заново с правильными данными.

## 📅 Расписание

### Можно ли сделать занятие раз в две недели?

**Нет, автоматически нельзя.**

Но можно использовать переносы:
```
/reschedule Никнейм дата дата новое_время
```

Или добавить напоминание себе вручную пропускать каждое второе занятие.

### Как удалить все расписание ученика?

Удаляй по одному дню:
```
/removeschedule Никнейм 0
/removeschedule Никнейм 1
/removeschedule Никнейм 2
...
```

Или удали ученика и добавь заново:
```
/remove Никнейм
```

## ⏰ Уведомления

### В какое время приходят напоминания?

- **О занятиях:** 3:00 утра (время Алматы)
- **Об оплате:** 10:00 утра (время Алматы)

### Как изменить время уведомлений?

Открой `complete_bot.py`, найди (в самом низу):

```python
scheduler.add_job(send_lesson_reminders, 'cron', hour=3, minute=0, args=[application])
scheduler.add_job(send_payment_reminders, 'cron', hour=10, minute=0, args=[application])
```

Измени `hour=3` на нужный час (0-23).

### Можно ли отключить уведомления?

**Да.** Закомментируй строки в `main.py`:

```python
# scheduler.add_job(send_lesson_reminders, 'cron', hour=3, minute=0, args=[application])
# scheduler.add_job(send_payment_reminders, 'cron', hour=10, minute=0, args=[application])
```

## 🔧 Технические вопросы

### На каком языке написан бот?

**Python 3.10+**

### Какие библиотеки использует?

- `python-telegram-bot` - работа с Telegram API
- `apscheduler` - автоматические задачи (уведомления)
- `pytz` - часовые пояса
- `sqlite3` - база данных (встроена в Python)

### Можно ли запустить на своем сервере?

**Да!** Бот работает везде где есть Python:
- VPS (DigitalOcean, AWS, и т.д.)
- Домашний компьютер (должен быть включен 24/7)
- Raspberry Pi

### Можно ли использовать PostgreSQL вместо SQLite?

**Можно**, но нужно переписать подключение к БД.

Для начала SQLite более чем достаточно (проверено на 100+ учениках).

## 🎓 PythonAnywhere

### Что такое PythonAnywhere?

Бесплатный хостинг для Python приложений.

**Преимущества:**
- ✅ Бесплатно навсегда
- ✅ Работает 24/7
- ✅ Не требует кредитной карты
- ✅ Простой интерфейс

**Ограничения бесплатного плана:**
- Нельзя устанавливать библиотеки не из белого списка (но все нужные есть!)
- Нужно продлевать раз в 3 месяца (1 кнопка)

### Что будет если не продлю аккаунт?

Через 3 месяца бот **остановится**.

Но **данные НЕ удалятся!**

Когда продлишь аккаунт - просто запусти бота заново, всё заработает.

### Можно ли запустить несколько ботов?

**Да!** На одном аккаунте PythonAnywhere можно запустить несколько процессов.

Просто загрузи файлы в разные папки с разными токенами.
